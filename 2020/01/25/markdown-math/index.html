使用 $ 包裹公式

使用 $ 或 $$ 包裹公式，被包裹的内容会全部展示为 LaTex 公式（使用 $$ 包裹的公式会独占一行且水平居中）

小贴士：字体控制，可以使公式更加美观，符号：\displaystyle
不使用字体控制的情况：$\frac{x+y}{y+z}$，效果：x+yy+z\frac{x+y}{y+z}y+zx+y​
使用字体控制的情况：$\displaystyle \frac{x+y}{y+z}$，效果：x+yy+z\displaystyle \frac{x+y}{y+z}y+zx+y​

括号、分隔符

  1. 下划线，符号：underline 或 <u> </u>
    如：$\underline{x + y + z}$，效果：x+y+z‾\underline{x + y + z}x+y+z​

  2. 上大括号，符号：\overbrace{算式}
    如：$\overbrace{a+b+c+d}^{2.0}$，效果：a+b+c+d⏞2.0\overbrace{a+b+c+d}^{2.0}a+b+c+d​2.0​

  3. 下大括号，符号：\underbrace{算式}
    如：$a+\underbrace{b+c}_{1.0}+d$，效果：a+b+c⏟1.0+da+\underbrace{b+c}_{1.0}+da+1.0b+c​​+d

  4. 上位符号，符号：\stacrel{上位符号}{基位符号}
    如：$\vec{x}\stackrel{\mathrm{def}}{=}{x_1,\cdots,x_n}$，效果：x⃗=defx1,⋯ ,xn\vec{x}\stackrel{\mathrm{def}}{=}{x_1,\cdots,x_n}x=defx1​,⋯,xn​

  5. 自适应括号，符号：\left \right
    在配对符号中可以让括号自动适应公式的高度，如：
    $\displaystyle \left( \frac{x}{2} \right)$，效果：(x2)\displaystyle \left( \frac{x}{2} \right)(2x​)

    另：在非配对符号中，使用 \left. 其他 \right符号 或 \left符号 其他 \right.（注意：有个 . ）

    如：$\displaystyle \left. \frac{du}{dx} \right|_{x=0}$，效果：dudx∣x=0\displaystyle \left. \frac{du}{dx} \right|_{x=0}dxdu​∣∣∣∣​x=0​

公式编号

注意：需编号的公式必须独占一行且居中，即必须使用 $$ 符号

常用 \tag{编号} 或 \tag*{编号}

编号 说明
\tag{编号} 公式宏包序号设置命令，可用于带星号公式环境中的公式行
\tag*{编号} 作用与\tag相同，只是标号两侧没有圆括号

例如：

$$ x2+y2=z^2 \tag{$1.1$} $$

$$ x2+y2=z^2 \tag*{$1.2$} $$

效果：

x2+y2=z2(1.1)x^2+y^2=z^2 \tag{$1.1$} x2+y2=z2(1.1)

x2+y2=z21.2x^2+y^2=z^2 \tag*{$1.2$} x2+y2=z21.2

数学运算

  1. 除法运算（分数表示），符号：\frac{分子}{分母} 或 分子 \over 分母
    如：$\frac{1-x}{y+1}$ 或 $x \over x+y$，效果：1−xy+1\frac{1-x}{y+1}y+11−x​ 或 xx+yx \over x+yx+yx​

  2. 平均数运算，符号：\overline{算式}
    如：$\overline{x + y + z}$，效果：x+y+z‾\overline{x + y + z}x+y+z​

  3. 开方运算，符号：\sqrt[开方数]{被开方数} （开二次方可直接使用 \sqrt ）
    如：$\sqrt x$，效果：x\sqrt xx​
           $\sqrt[3] y$，效果：y3\sqrt[3] y3y​

  4. 对数运算，符号：\log_低数{真数}
    如：$\log_5{8}$，效果：log⁡58\log_5{8}log5​8
