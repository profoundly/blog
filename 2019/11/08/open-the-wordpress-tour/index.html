当前设备屏幕尺寸过小，推荐使用PC模式浏览。

之前在 Shadowrocket 的电报群里看到有人分享了一个小火箭去广告的配置脚本，今天去 GitHub 上一看发现作者居然是西电的学长！之前就看到好多武大、重邮、东南的大佬的作品，用的 hexo 主题模版还是一个华科大佬写的，感觉他们在求学生涯里都能如此优秀，突然好后悔本科没有让自己 geek 起来，现在看到一个大佬是自己学校的就感到莫名的兴奋。

之前有想建一个自己的博客，先是用了 hexo，把内容都放到了 GitHub 上，不过访问速度有点慢，而且没找到好看的主题，打算寒假的时候搞一搞 wordpress，顺便把吃灰的服务器利用起来，现在先来占个坑 😄。
