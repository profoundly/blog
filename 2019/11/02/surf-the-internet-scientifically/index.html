当前设备屏幕尺寸过小，推荐使用PC模式浏览。

这里以 Centos 7 为例

ShadowsocksR

感觉 SS 和 SSR 正在慢慢被 V2ray 取代，这里就不多说了，直接上 teddySun 的一键脚本。

第一步是获得 root 权限，第二步是安装 ssr，第三步是安装 bbr 加速，如果无法安装就先 yum -y install wget。此外更详细的教程参考：https://github.com/allenking1028/ss/issues/1

V2ray

目前的 v2ray+ws+tls 简直稳的一批，基本不会被 q，就算被 q 了也可以套一层 cf 救活，所以还是推荐使用这个，不过前提是需要一个域名，像 example.domain.com 这样的三级域名也可以。

这里有两个一键脚本，我都用过，都可以用的。

第一步是获得 root 权限，第二、三步是安装 v2ray，选其一即可，第三步是安装 bbr 加速，如果无法安装就先 yum -y install wget。
