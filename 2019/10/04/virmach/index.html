VirMach — The Best & Cheapest Virtual Private Servers

VirMach 的 server 真滴便宜（不愧是“价格屠夫”），我入手的是最便宜的那个，一月 1.25 刀，一年 12.5 刀，之前搞活动的时候好像还有 7 刀一年的…

这是我第一次自己搭建 vps 来进行 surf the internet scientifically（阿里云不敢搭…），就目前来看网速还可以，youtube 最快也就 4000kbps，慢的时候几百 k（我是西安电信校园网 20M），基本每天都会时不时的抽风（难道 ssr 真的要凉了吗…等到期了还是试试 v2ray 吧），此外最烦的是每次 google 搜索的时候都会有个人机验证，好像便宜的 server 都会出现这种情况？算了，毕竟太便宜了，所以就先 shut up 8…

I Want to Say

因为之前用的是那种订阅的 vpn，每个月流量都是固定的，所以每过一阵就得看看还剩多少，免得月末不够用。查看的方式也很简单，就是登录网页自己去看，但是自从用了 virmach 之后，由于官网服务器在国外，访问的话挺费时间，而且登录前还有个 5s 等待时间，所以就想写个爬虫来得到剩余流量。（虽然一个月 500G，凭这网速也根本用不完，但是就是懒得点那几下鼠标…）

I’m so vegetable, 目前只是用了大半天时间搞了个雏形出来，打包成了 exe 文件，输入帐号、密码等个十秒就出来了（没办法…国外的服务器嘛）。其中可能会有 bug 或者可能爬不到，it’s all because i’m too green, but i will do it in time.
